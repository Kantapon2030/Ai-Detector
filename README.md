# 🛡️ พลิกเกมกลโกง (Ai-Detector)
**ระบบตรวจสอบความโปร่งใสและวิเคราะห์ความเที่ยงธรรมในการตอบคำถามด้วยพลัง AI**

โปรเจกต์นี้ถูกพัฒนาขึ้นเพื่อเป็นเครื่องมือช่วยอาจารย์และนักเรียนในการตรวจสอบว่า งานเขียนหรือเรียงความนั้นถูกสร้างขึ้นโดยมนุษย์ หรือใช้ AI (เช่น ChatGPT/Gemini) ในการเขียน โดยใช้เทคโนโลยี **RAG (Retrieval-Augmented Generation)** เพื่อให้ระบบสามารถ "เรียนรู้จากข้อผิดพลาด" และเก่งขึ้นเรื่อยๆ

## ✨ ฟีเจอร์เด่น (Key Features)
* **Gauge Meter Analysis:** แสดงผลลัพธ์เป็นเกจวัดความน่าจะเป็น (% Human vs % AI) ที่เข้าใจง่าย
* **Multimodal Support:** รองรับการอัปโหลดไฟล์ทั้ง ข้อความ (Text), PDF และรูปภาพ (PNG)
* **Localized Reasoning:** อธิบายเหตุผลเบื้องหลังการตัดสินใจของ AI เป็นภาษาไทยที่สั้นและเข้าใจง่าย
* **Learning Loop (Admin):** ระบบที่แอดมินสามารถแก้ไขการตัดสินของ AI และป้อนกลับเข้าสู่ฐานข้อมูลเพื่อให้ระบบจดจำเคสใหม่ๆ ได้ทันที

## 🛠️ Tech Stack
* **Frontend:** React 19, Tailwind CSS, Recharts
* **AI Engine:** Gemini 1.5 Flash (Google AI Studio)
* **Database & Auth:** Firebase (Firestore)
* **Intelligence Layer:** RAG Implementation with Vector Simulation

## 🚀 เริ่มต้นใช้งาน
1. วางข้อความหรืออัปโหลดไฟล์ที่ต้องการทดสอบในหน้าหลัก
2. ระบบจะวิเคราะห์และแสดงผลทันทีผ่าน Gauge Meter
3. อ่านเหตุผลประกอบที่ด้านล่างของผลลัพธ์
